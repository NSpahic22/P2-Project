TO USE THE PROGRAM:
The program is meant to be used via Visual Studio Code

1. Check if you have "Live Server" extension installed in Visual Studio code. If not, follow this guide: 
1.2 Open VSCode and type ctrl+P, type ext install ritwickdey.liveserver.

2. Open Visual Studio Code and click on "File" in top left corner.
3. On the drop down menu click on "Open Folder" and select the extracted version of "P2-Project".
4. Right click "hjemmeside.html" and click on "Open with Live Server"
5. Your default browser should now load the site
